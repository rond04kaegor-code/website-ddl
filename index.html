DDLanguage

Debian Distributions Language — проще Python в 10000 раз
GPLv3 Linux native

Минималистичный язык программирования для Linux. Писать код должно быть так же просто, как говорить.

🚀 Почему DDL

📖 Синтаксис без лишнего

Никаких точек с запятой, сложных импортов. Код читается как обычный текст.

⚡ Мгновенная компиляция

.ddl файл за секунды становится нативным бинарником. Чистый C++ на выходе.

🖥️ Встроенная графика

Окно с кнопками — одной строкой. Браузер — три строки. GTK3 и WebKit под капотом.

📦 Экспорт в .deb

Написали программу — собрали установочный пакет. Распространяйте как обычные deb-пакеты.

👥 Для кого

Для тех, кто устал от сложных языков. Для начинающих, которые хотят результат сразу. Для опытных, кому нужен быстрый инструмент под Linux.

⚙️ Одна команда

cppddl

Установка, компиляция, запуск и экспорт — всё через единственную программу. Никаких виртуальных окружений.
